Para calcular a Área do Triangulo
é necessario a Fórmula: A = (lado² * √3) / 4

    Passo a Passo:
  • Meça o comprimento de um dos
    lados do triângulo.
  • Eleve esse valor ao quadrado.
  • Multiplique o resultado por √3.
  • Divida o resultado por 4 para
    obter a Área.
Área do Circulo

{% if area_c is defined %} Área do Circulo = {{ area_c }} {% else %} {% endif %}

Para calcular a Área do Circulo
é necessario a Fórmula: A = πr²
onde π é aproximadamente 3,14

    Passo a Passo:
  • É necessario encontrar o comprimento do raio do círculo.
  • Se tiver o valor do diâmetro (d), pode encontrar
    o raio dividindo o diâmetro por 2
  • Substitua o valor do raio na fórmula e descubra a Área.
